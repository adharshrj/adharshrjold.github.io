Sleep Shift Scheduling Tool

An intelligent application that helps astronauts with sleep scheduling, nutrition and exercise.

ROLE:-
I performed a complete analysis on the dataset provided by NASA to derive the essential parameters. Based on my research we designed a preliminary web application. Then I ported the web app into an android app using Android SDK. We cross checked the parameters and submitted after finalization. I wanted the application to be as light as possible so that we can port them to smart watches in the future.

LANGUAGES USED:-
Python, JSON, HTML, CSS, Java, Kotlin
